# tachimanga

A filtered Tachiyomi extension repository containing only:

- **MangaFire**
- **Sushi-Scan**
- **ReadComicOnline**

## Setup

Add the following URL as a custom extension repository in Tachimanga:

```
https://raw.githubusercontent.com/Adonan17/tachimanga/main/index.min.json
```

The index is updated automatically every 12 hours via GitHub Actions.
